# Briefmarken-Matrixcode-Scanner

Einfache lokale Vite-Webanwendung zum Scannen von Datamatrix-Codes auf Briefmarken von deutschepost.de, um die Sendungsnummer ohne offizielle App auszulesen.

## Entwicklung

```bash
npm install
npm run dev
```

## Produktion

```bash
npm run build
npm run preview
```
